Portfolio 4: Make Your Own Portfolio

Deformation involves changing the appearance of an object or shape. This change is done using control points that surround the object or shape. Algorithms then compute how the surrounded object or shape should change based on the positions of these control points.

Deformation can occur in both 3D and 2D contexts and is often used in computer animations. A well-known method for 3D deformation is harmonic coordinates, which was developed for Pixar character animations. These 3D deformations use "cages" of control points around the characters to control their movements.

We will be showing 2D deformations, using a method known as lattice deformation. Doing deformation in 2D will be simpler than doing deformation in 3D, allowing us to focus on the main concepts of deformation.

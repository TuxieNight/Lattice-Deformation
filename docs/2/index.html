Page 2: 2D Animated Deformation with Lattices

Lattice deformation can be useful for creating animations. By gradually changing the positions of the control points, we can create smooth transitions between different states. Additionally, the animations can easily be updated when using deformation, because they only require changing a few parameters. It is much easier to animate a few control points than it is to animate the entire shape. This is especially true for complex shapes, where animating the entire shape would be very difficult. By using lattice deformation, we can create smooth animations with minimal effort.

This is a demonstration of 2D animated deformation using lattices.

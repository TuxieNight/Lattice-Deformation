Page 4: Multiple Lattice Deformations & Shapes

This is a demonstration of the flexibility of deformation. You can give each shape its own lattice, where each lattice can be controlled independently. You can also use the same lattice for multiple shapes.

These concepts are very useful. Think about animating a character. You might want to bend the arm at the elbow. This would require moving the shape that forms the skin and the shape that forms the sleeves at the same time. In this case, it is useful to have both deformed by the same lattice. However, you probably would not want the deformation of the arm to affect the deformation of the leg. In this case, it is useful to have separate lattices for each shape.

This change required setting up the structures for the lattices and shapes differently, to allow for the flexibility of having any number of lattices and any number of shapes.

You can use the checkbox to toggle the visibility of the grid.

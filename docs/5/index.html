Page 5: Articulated Motion

This is a demonstration of applying deformation to create articulated motion for character modeling.

An alien character is drawn using many shapes. Shapes that are related (like the face, eyes, and mouth) are deformed with the same lattice. This allows for the shapes to move together, creating nice behavior. You can try changing the shape of the alien's face and watch as it does not affect the body's shape.

Similarly, you can see that shapes can be deformed with lattices so that they look more like the shape you want. This is evidenced by the lattices of the arms.

Additionally, shapes that are mostly unrelated (like the head and the body) should have different, but not entirely independent, lattices. This is because the parts of a body, though able to move separately, are still attached together and so should affect each other somewhat. You can introduce hierarchical structures to enable one lattice to influence another. In this case, the head and the arms, being attached to the body, should move with it. This allows for the alien to have a more natural motion, as the body parts can move independently but still interact with each other. See this in action when you move the central (pale purple) control point of the body.

You can also start and pause an animation of the alien. (Note that you cannot drag points while the animation is underway!) Observe how the lattices move.

  • The body also moves up and down, and all of the body parts move with it.
  • The body expands and contracts, like the animation from page 2, letting the alien breathe.
  • Two control points of each arm are "anchored" to the shoulder so that the arm doesn't "run off" the body as the rest of the arm waves.
  • The central point of the head's grid (which influences the four surrounding points) is moved, moving the eyes and the mouth but leaving the circular boundary minimally affected.

Page 1: 2D Deformation with Lattices

The below is a demonstration of 2D deformation using lattices.

Lattice deformation uses a grid of control points to deform a shape. The control points are used to determine the position of the shape's points. The points of the shape are affected by the four closest control points (the control points of the grid cell they belong to). Each point remembers its undeformed position relative to its control points and uses this relative position to determine its new position when the control points are moved. We convert from relative position to absolute position using bilinear interpolation.

Click and drag the points to deform the grid. Use the slider to change the number of control points. Use the checkbox to toggle the visibility of the grid.

The heart itself is made of multiple points connected with lineTo().
The heart's points are moved when the control points are dragged, causing a change in the direction of the lines and overall deformation of the shape.
Each of the heart's points are only affected by the 4 closest control points.

Number of Control Points

Notice how the number of control points affects the amount of ease and control you have over the shape. With more control points, you have more control over the shape, but it is harder to use the lattice. For example, the more control points you have, the more points you have to drag to get the desired shape. Additionally, more control points can make it harder to avoid undesired sharp edges. With fewer control points, it is easier to use the lattice, but you have less control over the shape. The number of control points is a tradeoff between ease of use and control over the shape.

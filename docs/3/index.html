Page 3: Lattice Deformations with Simplified UI

Now it is your turn to test out animating with lattice deformation. This page not only demonstrates the benefits of animating with lattice deformations, but also shows how a simplified UI can make the process easier. This demonstration gives the user some additional control points that simplifies the process of deforming an object to a desired shape.

The additional (blue) control points affect the four points of the grid cell they belong to, effectively making common large-scale changes easier. The actual grid control points can still be altered to make more precise changes.

The animating feature transitions between the two states the user sets. The additional (blue) control points are useful, because they make make getting the desired states much easier.

By default, state1 and state2 are both set to be the initial, undeformed heart position. To change one of the states to a certain grid setup, first deform the grid to the desired setup. Then, click the button of the desired state to change. After setting up the desired states, checkmark the transition checkbox to see the animation between the two states. The speed of the animation can be adjusted with the slider below.

As usual, users can get a better view of their beautiful animation by setting the grid to be invisible.

Again, note how using lattice deformation makes it easier to animate the shape. The two states only have to store the positions of the control points, rather than the positions of all the points in the shape. This is beneficial storage-wise and also makes animating simpler. Additionally, the animation can be easily updated by changing the control points, rather than having to change each individual point of the shape.
